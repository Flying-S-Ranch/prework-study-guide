Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Child elements are nested within parent elements
  • Head elements are usually not seen by people looking at the web page
  • Within the body it is important to order and nest things neatly so that the finished web page is visually clear
  • Elements are opened and closed with 'tags'
  • The opening tag is held within angle brackets
  • The closing tag is the same as the opening tag, with a forward slash just inside the left angle bracket

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content iside an element
  • CSS code adds style to the HTML code that builds a webpage
  • CSS code can be added directly into HTML code, but this can be difficult to edit
  • A separate 'style guide' can be built in CSS allowing changes to be made more easily
  • Separate CSS code must be linked into the HTML code to be applied

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git bash is where you enter code on your desktop
  • git hub is an online resource to share and collaborate on code
  • cd: change directory one folder at a time
  • cd ..: go back one folder at a time
